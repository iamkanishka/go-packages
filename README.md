## 🎯 Purpose

- 📌 Central place to discover all Go SDKs
- 🔗 Keep each SDK independent and modular
- 🚀 Improve ecosystem visibility
- 🧩 Enable easy exploration of APIs

---

## 🧠 Philosophy

Each SDK is:

- ✅ A standalone Go module
- ✅ Independently versioned
- ✅ Focused on a single provider
- ✅ Designed for real-world usage

This ensures flexibility, scalability, and maintainability.
